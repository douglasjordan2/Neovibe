“Welcome Home”

You already know this

Neovibe doesn't drop you into a blank terminal. You get a dashboard, quick actions, and recent files — like opening VS Code.

“Find Your Way”

Navigation for VS Code users

Cmd+P? We have that. File explorer? Got it. But wait until you see Oil — a file manager inside your buffer.

<leader>ff — that's your Cmd+P

“Discover as You Go”

Which-key + the basics

Press leader and wait. Everything reveals itself. No memorization needed.

Visual select + J/K to move lines. <leader>y to yank to system clipboard.

“Speed of Thought”

Power moves

This is where Vim users live. Flash jump. LSP everywhere. Git in your gutter.

“AI, Not a Plugin”

The main event

Claude is built into Neovibe. Chat, inline edit, diff review — no API key, just your subscription.

Install

Prerequisites

Neovim ≥ 0.10 • Claude Code CLI • Git • Nerd Font

Quick start

git clone https://github.com/douglasjordan2/neovibe ~/.config/neovibe
NVIM_APPNAME=neovibe nvim

Alias (optional)

# Add to your .bashrc / .zshrc
alias neovibe='NVIM_APPNAME=neovibe nvim'
